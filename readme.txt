PrismPay Plugin extends the functionality of WooCommerce to accept payments from credit/debit cards and e-checks on your checkout page using PrismPay Integration.

== Installation ==

1. Download the plugin zip file.
2. Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
3. Click on the "Upload" option, then click "Choose File" to select the plugin zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4. Activate the plugin.
5. Open the settings page for WooCommerce and click the "Payment Gateways" tab.
6. Click on the sub tab for "Prismpay" and configure your Prismpay Gateway settings.

== Description ==

<h3>PrismPay Plugin</h3> for WooCommerce makes your website ready to use PrismPay Payment Gateway, to accept credit/debit cards and e-checks on your e-commerce store checkout page.

<h3>WooCommerce 2.x Compatible</h3>

== Features ==

1. Accept Credit/Debit Cards and e-checks right on your website.
2. No redirecting on other url.
3. Easy to install and configure
4. Option to configure success & failure message

== FAQ ==

Do I need a merchant account before I can use the Prismpay Payment Gateway plugin?

Yes. In order to use this plugin you will need a merchant services account. For more information, please visit: http://www.prismpay.com/.

What is the cost for the Prismpay Payment Gateway plugin?

This plugin is a FREE download.

Does the plugin credit card and e-check processing?

Yes. This plugin support credit card and e-check processing.
